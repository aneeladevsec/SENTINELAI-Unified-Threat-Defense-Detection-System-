## 📞 How to Use the System

### Quick Start (5 minutes)
```bash
# 1. Access Dashboard
Open: http://localhost:8501

# 2. View API Documentation
Open: http://localhost:8000/docs

# 3. Run Tests
python test_api.py

# 4. Test Alert Creation
curl -X POST http://localhost:8000/alerts \
  -H "Content-Type: application/json" \
  -d '{"alert_type":"test","severity":"high","description":"Test"}'
```

### API Integration Example
```python
import requests

# Create alert
response = requests.post('http://localhost:8000/alerts', json={
    'alert_type': 'dos_ddos',
    'severity': 'high',
    'confidence': 0.92,
    'source_ip': '203.0.113.42',
    'destination_ip': '192.168.1.100'
})
alert_id = response.json()['alert_id']

# Evaluate threat
response = requests.post('http://localhost:8000/defense/evaluate', json={
    'alert_id': alert_id,
    'attack_type': 'dos_ddos',
    'confidence': 0.95
})
threat_level = response.json()['threat_level']

# Get risk forecast
response = requests.get(f'http://localhost:8000/prediction/risk/asset_001')
risk_score = response.json()['risk_score']
```
